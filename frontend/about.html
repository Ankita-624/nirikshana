About Nirikshana

Problem

Water bodies face rising pollution; current monitoring is slow, manual, and lab-dependent. Communities and authorities need fast, low-cost, real-time visibility.

Solution

Low-cost IoT sensor stations stream multi-parameter data (pH, turbidity, TDS, temperature, DO) to a backend that classifies status and raises alerts. A public dashboard shows live pins, KPIs, and station history.

End-to-End Pipeline

  1. Edge capture (ESP32 + sensors) → timestamp, basic preprocessing.
  2. Ingest API (HTTP/MQTT) → storage (time-series DB).
  3. Classification/thresholds → alerting.
  4. Dashboard (map, KPIs, charts) + CSV export.

Stakeholders & Impact

Water Resources Dept., Environmental Agencies, Researchers, Communities. Benefits: faster detection & response, transparency, and scalable coverage.
